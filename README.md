# Online Retail Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics pipeline using Python and PostgreSQL.
The dataset contains transactional data from an online retail store.

## Tech Stack
- Python
- pandas
- SQLAlchemy
- PostgreSQL
- Jupyter Notebook

## Project Structure

## Workflow
1. Exploratory Data Analysis (EDA) using Jupyter Notebook
2. Data transformation and feature engineering
3. Load cleaned data into PostgreSQL
4. Enable analytics and reporting

## Key Transformations
- Converted InvoiceDate to datetime
- Created TotalPrice feature
- Identified cancelled transactions
- Extracted time-based features

## Example Analytics Questions
- Total revenue by country
- Monthly sales trends
- Refund rates over time
